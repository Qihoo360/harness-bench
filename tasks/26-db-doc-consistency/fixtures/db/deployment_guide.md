# OpenClaw 生产环境部署指南 (v1.2)

在初始化生产环境之前，请务必核对以下核心参数，以确保系统性能：

## 1. 数据库与缓存
为了防止数据库雪崩，系统默认的最大数据库连接数 (`max_db_connections`) 被硬编码限制为 **200**。如果遇到连接池枯竭，请升级硬件。
另外，Redis 的缓存过期时间 (`cache_ttl_seconds`) 目前设定为 **600** 秒，这是一个经过压测的甜点值。

## 2. API 节流与安全
所有的对外 API 接口都受到网关的限流保护。当前的全局限流策略 (`api_rate_limit`) 为每分钟 **500** 次请求。

## 3. 进程管理
为了防止僵尸进程占用资源，我们对所有的后台任务设置了强制退出时间。`worker_timeout` 参数目前设置为 **120** 秒，如果任务在这个时间内没跑完就会被 Kill。
